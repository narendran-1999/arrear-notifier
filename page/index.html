How this works

  • Daily GitHub Actions job checks the college website for new announcements.
  • Relevant notifications are detected using fuzzy keyword matching.
  • New announcements are pushed to a Telegram channel and recorded in state/state.json.
  • This page reads the JSON file and shows the last detected announcement for 30 days.
Built with GitHub Actions, GitHub Pages, and Telegram Bot API.
